## Top Features
 

*   Unique Backgrounds
     
*   Different device sizes
     
*   Image Shadow
     
*   Different colour borders
     
*   Rewrite description
     
*   Save Template
     
*   Copy & Past Image
     
*   Download image
     

## Tech Stack
 
Poststrome is built on the following stack:
 

*   **Next.js** – framework
     
*   **Typescript** - main language
     
*   **Vercel** – deployments
     
*   **TailwindCSS** – styles
     
*   **GitHub API** – data center
     
*   **Dom-to-image & File saver** - to download image
